# OVH AI Chat

A simple AI chat application using OVH AI Endpoints.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set your OVH AI API key:
   ```bash
   export OVH_AI_API_KEY=your_api_key_here
   ```

3. Build the project:
   ```bash
   npm run build
   ```

4. Run the chat:
   ```bash
   npm start
   ```

## Usage

Type your messages and press enter. The AI will respond. Type "exit" to quit.

## Note

Make sure to replace the BASE_URL with the correct one for your model and region. Check OVH documentation for available endpoints.
